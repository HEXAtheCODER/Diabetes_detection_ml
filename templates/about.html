About

The above project is done by the help of Logistic Regression algorithm. This logistic regression is a part of scikit-learn libary. Scikit-learn (formerly scikits.learn and also known as sklearn) is a free software machine learning library for the Python programming language. It features various classification, regression and clustering algorithms including support vector machines, random forest etc., and is designed to interoperate with the Python numerical and scientific libraries NumPy and SciPy. The web application is made by the help of HTML, CSS, Bootstrap and the backend is done by the help of Flask, a micro wed framework written in flask.
This project is deployed on Heroku.
